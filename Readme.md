Na luta CockMon, o Cockmon com maior nível/Level ganhará. O Cockmon ganhador ganhará uma quantidade X de experiência, e o perdedor
ganha uma quantidade Y de experiência, sendo Y > X. Logo, o Cockmon perdedor ganha uma vantagem de experiência em realção ao
ganhador. Porque o DEV não quer pensar sobre mecânica de combate.
